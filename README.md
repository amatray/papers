# Adrien Matray Papers

Public PDF hosting for papers linked from Adrien Matray's website.

## How to Add a Paper

1. Add the PDF to the `papers/` folder.
2. Commit and push the change.
3. Link to the PDF using:

```text
https://amatray.github.io/papers/papers/FILENAME.pdf
```

For example, a file saved as `papers/my-paper.pdf` will be available at:

```text
https://amatray.github.io/papers/papers/my-paper.pdf
```
